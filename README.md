# BDIX Live TV Server Application

## Overview
This repository contains the source code and documentation for the BDIX Live TV server application, which is designed to stream live television channels over the BDIX infrastructure.

## Features
- Live streaming of HD channels
- Multichannel support
- User authentication and access control
- Real-time monitoring of streaming quality

## Installation
To install the BDIX Live TV server application, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/shamimpipon/Shamim-Live-TV-Update.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Shamim-Live-TV-Update
   ```

3. Install the required dependencies:
   ```bash
   npm install
   ```

4. Configure the application by editing the `config.json` file:
   ```json
   {
     "serverPort": 3000,
     "dbConnectionString": "mongodb://localhost:27017/bdixlivetv"
   }
   ```

5. Start the server:
   ```bash
   npm start
   ```

## Usage
After starting the server, you can access the application by navigating to `http://localhost:3000` in your web browser.
